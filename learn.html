🎨 Meet Fidget Creator

Fidget Creator is your friendly tool for crafting .fidget files— interactive visual-sound widgets designed to spin, toggle, and delight.

🛠 Features

  • Toggle and Spinner types
  • ImageA + optional ImageB & Sound
  • Live JSON preview while you edit
  • Automatic image resizing to 256×256 (or turn it off!)
  • Preview spinner with angular velocity in rad/s
  • Downloadable .fidget files with embedded data

✨ Advanced Mode

Need more control? Flip the Advanced Mode switch to unlock additional options:

  • Disable auto-resizing
  • Keep your original file format
  • Edit the schemaVersion
💡 Tip: Images are auto-scaled and converted to PNG by default— unless Advanced Mode says otherwise!

📁 The .fidget Format

Every .fidget is a JSON file containing your metadata and assets encoded as Data URLs. For example: 

{
  "type": "spinner",
  "name": "My Cool Spinny",
  "imageA": "data:image/png;base64,...",
  "velocity": 40
}

You can create .fidget files without ever touching code—just upload, preview, and export.
